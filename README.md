# ROB_vis_aruco
Tracking of aruco markers

1) Open a terminal

    Navigate to the PATH where the reposetory is:

    a)

    ``` bash
    cd PATH/
    ```

    b)

    run the python script

    ``` bash
    python3 robVis_pkg/robVis_node.py
    ```
